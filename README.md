# DAS

Description
---------

Dune : Arrakis Smuggler

Jeu de gestion/survie basé sur l'univers du roman Dune, écrit par Franck Herbert.

Prérequis
---------

- Java (jdk-21.0.4.7)
- Git

Installation
------------

Cloner le dépôt :

```bash
git clone https://github.com/suFredericD/DAS.git
```
