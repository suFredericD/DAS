# DAS

Description
===========

Ce dépôt contient le projet DAS (Démo / Application / Service). Il sert de point de départ pour développer et versionner le code dans ce workspace.

Prérequis
---------

- Java (version selon le projet)
- Git

Installation
------------

1. Cloner le dépôt :

```bash
git clone https://github.com/suFredericD/DAS.git
```

2. Se placer dans le dossier :

```bash
cd DAS
```

Utilisation
-----------

- Compiler / lancer selon la configuration du projet (ex: `mvn clean install`, `gradle build`, etc.).
- Ajouter des instructions spécifiques ici selon la stack.

Contribuer
----------

- Ouvrir une branche pour chaque fonctionnalité : `git checkout -b feature/ma-fonction`.
- Faire des commits clairs et pousser vos branches.
- Ouvrir une pull request sur GitHub pour revue.
